# Get support

## Cloud Customer Care offerings

Support offerings are available from Google Cloud that can include 24/7 coverage, phone support, and access to technical account services. For more information, see [Getting Cloud Customer Care](/support/docs) .

## Request quota changes

To increase or decrease quotas, see [Quotas and limits](/bigquery/quotas) .

## Ask a development question

For help using the BigQuery API you can post questions to the developer community on [Stack Overflow](https://stackoverflow.com/questions/ask?tags=google-bigquery) with the tag [google-bigquery](https://stackoverflow.com/questions/tagged/google-bigquery) .

  - [Ask a question](https://stackoverflow.com/questions/ask?tags=google-bigquery)
  - [See a list of all questions](https://stackoverflow.com/questions/tagged/google-bigquery)

## Discuss BigQuery

Visit the Google Cloud [Slack community](https://googlecloud-community.slack.com/) to discuss BigQuery and other Google Cloud products. If you haven't already joined, [use this form to sign up](https://join.slack.com/t/googlecloud-community/shared_invite/zt-3icuwfdvq-eq_58LqmOoCNc16mpZcaNA) . For BigQuery, join the [`  #bigquery  `](https://googlecloud-community.slack.com/messages/C0E2CFHJT/) channel.

## File feature requests and bug reports

If you'd like to request a new feature for BigQuery, or encounter a reproducible bug, you can report them through the [issue tracker](/support/docs/issue-trackers) . You can [review the open issues](https://issuetracker.google.com/savedsearches/559654) to see if your bug or request has already been reported, and star the existing thread to keep updated. If there's no existing thread, you can [request a new feature](https://issuetracker.google.com/issues/new?component=187149&template=1162685) or [create new bug](https://issuetracker.google.com/issues/new?component=187149&template=0) .
