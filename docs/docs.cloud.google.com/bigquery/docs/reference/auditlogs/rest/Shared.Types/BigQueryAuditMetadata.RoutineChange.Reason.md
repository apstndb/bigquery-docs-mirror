Describes how the routine was updated.

Enums

`REASON_UNSPECIFIED`

Unknown.

`QUERY`

Routine was updated using a DDL query.

`ROUTINE_UPDATE_REQUEST`

Routine was updated using the routines.update or routines.patch API.
